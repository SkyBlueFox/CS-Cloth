## Frontend

`frontend/` is the SvelteKit application for CS Cloth.

Current role:
- Public storefront
- Customer account, cart, checkout, wallet, and order screens
- Admin and superadmin dashboards

Runtime config:
- `frontend/.env` should define `BACKEND_URL`
- In Docker, the default is `http://laravel-api`

Use the repository root [README.md](/Users/Tan/Uni/WebTech/CS-Cloth/README.md) for setup and run instructions.
