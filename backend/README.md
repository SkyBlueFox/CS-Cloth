## Backend

`backend/` is the Laravel 12 API for CS Cloth.

Current role:
- Serves JSON endpoints from `routes/api.php`
- Sends OTP emails with `resources/views/emails/otp-code.blade.php`
- Does not serve the application frontend anymore
